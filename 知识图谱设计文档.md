## 一、本体结构

**7 类节点，7 类关系。**

| 节点 | 标签 | 说明 | 关键属性 |
|------|------|------|----------|
| 故障现象 | `Symptom` | 设备异常状态或报警 | `primary_features`(String[]), `secondary_features`(String[]), `sensory_type`(String[]) |
| 故障原因 | `Cause` | 导致现象的根本原因 | `check_method`(visual/measure/disassemble), `check_time`, `requires_shutdown`, `failure_rate`(0~1) |
| 维修步骤 | `Step` | 单个修复操作 | `duration`, `min_personnel`, `safety_level`(normal/caution/danger) |
| 设备 | `Equipment` | 船舶电气设备 | `level`(2/3) |
| 备件 | `SparePart` | 维修所需零部件 | — |
| 工具 | `Tool` | 检测仪表或工具 | `functions`(String[]), `alternatives`(String[]) |
| 注意事项 | `Precaution` | 安全警告或禁忌 | `severity`(forbidden/warning/info), `consequence`, `consequence_type` |

| 关系 | 方向 | 说明 |
|------|------|------|
| `CAUSED_BY` | Symptom → Cause | 关系属性: `priority`(Int), `context`(String) |
| `FIXED_BY` | Cause → Step | 指向修复第一步（单步=唯一步骤，多步=第一步） |
| `NEXT_STEP` | Step → Step | 多步流程顺序链。单步无此关系 |
| `BELONGS_TO` | Symptom → Equipment | 现象归属设备 |
| `USES_SPAREPART` | Step → SparePart | 步骤需要的备件 |
| `REQUIRES_TOOL` | Step → Tool | 步骤需要的工具 |
| `HAS_PRECAUTION` | Step → Precaution | 步骤的安全注意事项 |

**用户查询路径（单一入口 → 完整输出）：**

```
用户输入: "电动机不能起动"
  ↓
Symptom → CAUSED_BY (ORDER BY priority) → Cause (含 check_time/check_method)
  ↓
FIXED_BY → Step₁ → NEXT_STEP → Step₂ → NEXT_STEP → Step₃
  ↓                          ↓                ↓
REQUIRES_TOOL → Tool    REQUIRES_TOOL    USES_SPAREPART → SparePart
HAS_PRECAUTION → Prec   HAS_PRECAUTION   HAS_PRECAUTION → Prec
  ↓
source_page → "第五章第三节" (可信度校验引用)
```

**设计决策：**

- `priority` 存于 `CAUSED_BY` 关系边——同一 Cause 可被多 Symptom 共享，不同 Symptom 下优先级不同
- `NEXT_STEP` 定义步骤顺序——插入/删除步骤只改关系，不重编号
- 所有节点携带 `source_doc` + `source_page`，用于可信度校验
- 无 RepairProcedure 中间层——赛题未要求，90% Cause 仅需单步，中间层无增益

## 四、多步流程

以下 Cause 修复需多步。导入结构：`Cause → FIXED_BY → Step1 → NEXT_STEP → Step2 → NEXT_STEP → Step3`

| Cause | Step1 | Step2 | Step3 |
|-------|-------|-------|-------|
| 定子绕组断路 | 电桥测各相电阻 | 找断路点 | 修复或重绕 |
| 定子绕组短路 | 电桥测电阻 | 短路侦察器逐槽定位 | 加热翻出包绝缘重嵌 |
| 定子绕组接地 | 绝缘电阻表测对地 | 解体目视/校验灯/升压法定位 | 软化垫绝缘衬垫 |
| 绕组接线错误 | 感应法判首尾 | 指南针查极相组方向 | 纠正接线 |
| 转子断笼 | 短路侦察器逐槽查 | 定断条位置 | 焊接/冷接/换条 |
| 电动机噪声大 | 切电源判来源 | 电磁→查绕组 机械→查轴承 | 对症修复 |
| 电动机振动大 | 切电源判来源 | 电磁→查绕组 机械→查联轴器 | 对症修复 |
| 电刷不在中性线 | 感应法接毫伏表 | 按按钮移刷架 | 固定 |
| 电枢脱焊 | 电桥逐片测 | 找异常偏大两片 | 重焊 |
| 相复励坏 | 查整流器 | 查移相电抗器 | 查复励变压器 |
| VT全导通 | 查VT击穿 | 查CF误触发 | 查BJ/CL |
| VT无触发 | 查CF | 查BJ开路 | 查CL |
| 总线无登记 | 断总线测电压 | 8~12V=短路/否=控制器坏 | 修复 |
| 无信息异常 | 模块分割法 | 分析法+图纸I/O表 | 插拔/交换锁定 |
| 点不着火 | 查火花 | 查喷油 | 查风门及2JZ |
| 变压器绕组短路 | 停机断开关目视 | 万用表定位 | 修或重绕 |
