# 工程开发注意事项（双人协作）

> 项目：船舶装备故障诊断智能问答系统  
> 技术栈：Python FastAPI + Neo4j + LangChain + Vue3 + Electron

---

## 一、环境隔离

```bash
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

`requirements.txt` 锁定版本。`.env` 文件不入 Git。

---

## 二、Neo4j 连接

```python
# ✅ 从环境变量读取
driver = GraphDatabase.driver(
    os.getenv("NEO4J_URI", "bolt://localhost:7687"),
    auth=(os.getenv("NEO4J_USER", "neo4j"), os.getenv("NEO4J_PASSWORD"))
)
```

---

## 三、API 契约

| 方法 | 路径 | 成员 | 说明 |
|------|------|------|------|
| GET | `/health` | A | 健康检查 |
| GET | `/symptoms` | A | 返回所有故障现象名称 |
| POST | `/ask` | A | 问答接口（SSE 流式） |
| POST | `/parse_document` | B→A | 上传文档，返回候选三元组 |
| POST | `/confirm_import` | B→A | 确认候选三元组入库 |
| GET | `/export_sync` | A | 导出知识包到 U盘 |
| POST | `/import_sync` | A | 从 U盘导入知识包 |

## 六、开发原则

1. **环境变量**：密码/密钥一律从 `os.getenv()` 读取，禁止硬编码
2. **CORS**：后端必须配置 CORS 中间件
3. **Neo4j session**：用 `with driver.session() as session:` 保证自动关闭
4. **Cypher 参数化**：禁止字符串拼接，用 `$param` 占位符
5. **Git**：feat/xxx 分支，`feat: 描述` 提交信息，提交前 `git pull --rebase`
6. **知识图谱备份**：每次大改前先 Dump 数据库
