# 知识图谱与图检索

## 一句话解释

知识图谱 = 用**节点和连线**表示知识，可以沿着连线**推理**。

## 类比

社交网络就是最简单的知识图谱：
```
[张三] ──朋友──→ [李四] ──同事──→ [王五]
  │                                    │
  └────────── 间接认识 ────────────────┘
```

你不需要认识王五，但通过"张三→李四→王五"这条路径能推断出关系。

## 在我们的项目里

```
[燃油泵] ──属于──→ [燃油系统] ──属于──→ [发动机]
  │
  ├──故障原因──→ [密封圈老化]
  ├──故障原因──→ [叶轮磨损]
  ├──维修步骤──→ [更换密封圈]
  └──注意事项──→ [更换前释放燃油压力]
```

用户问"发动机漏油怎么办"时：
1. 实体识别：提取"发动机""漏油"
2. 图查询：`发动机 → 包含哪些零件 → 哪些零件会漏油 → 什么原因 → 怎么修`
3. 图返回结构化结果：燃油泵密封圈老化 → 更换密封圈，油管接头松动 → 紧固接头

## 图检索 vs 文本检索

| | 文本检索（向量/BM25） | 图检索（Neo4j） |
|------|---------------------|----------------|
| 输入 | 任意文本 | 实体名称或关系 |
| 输出 | 相关文档片段 | 结构化三元组 |
| 优势 | 覆盖面广，什么都能搜 | 精确关联，可推理 |
| 局限 | 不理解实体间的关系 | 只能搜已结构化的知识 |

## Neo4j 是什么

Neo4j 是目前最流行的图数据库。它用 **Cypher** 语言查询。

### Cypher 查询示例

```cypher
-- 查询：发动机有哪些常见故障？
MATCH (e:Equipment {name: '发动机'})-[:HAS_FAILURE]->(f:Failure)
RETURN f.name, f.description

-- 查询：燃油泵漏油怎么修？
MATCH (p:Part {name: '燃油泵'})-[:HAS_FAILURE]->(f:Failure {type: '漏油'})
MATCH (f)-[:FIXED_BY]->(s:Solution)
RETURN s.steps, s.cautions
```

### 类比：SQL vs Cypher

| SQL（传统数据库） | Cypher（图数据库） |
|------|------|
| SELECT ... FROM ... WHERE ... | MATCH (节点)-[关系]->(节点) WHERE ... |
| 适合：表格数据 | 适合：关联数据 |
| 例：查用户订单 | 例：查故障原因链 |

## 图检索在 RAG 中的位置

```
用户问题 → 实体识别 → 提取实体名（如"燃油泵"）
                         │
                         ├─→ Neo4j Cypher 查询 → 结构化三元组
                         │                              │
                         └─→ 同时 Embedding → 向量检索  │
                                               │         │
                                               └────┬────┘
                                                    ↓
                                              融合排序 → Prompt
```

图检索的结果（三元组）以文字形式注入 prompt，例如：
```
知识图谱检索结果：
- 燃油泵 → 故障原因 → 密封圈老化
- 燃油泵 → 维修步骤 → (1)断开电源 (2)释放燃油压力 (3)更换密封圈...
- 燃油泵 → 注意事项 → 更换前必须释放燃油压力
```

## 我们项目的知识图谱本体（初步）

| 实体类型 | 例子 | 属性 |
|----------|------|------|
| Equipment（设备） | 发动机、液压系统、电气系统 | 型号、功率 |
| Part（零部件） | 燃油泵、密封圈、叶轮 | 编号、规格 |
| Failure（故障现象） | 漏油、异响、无法启动 | 严重程度、频率 |
| Cause（故障原因） | 密封圈老化、叶轮磨损 | 概率 |
| Solution（维修步骤） | 更换密封圈 | 步骤序号、工具 |
| Caution（注意事项） | 释放燃油压力 | 安全等级 |

| 关系类型 | 方向 | 例子 |
|----------|------|------|
| HAS_PART | Equipment→Part | 发动机→燃油泵 |
| HAS_FAILURE | Part→Failure | 燃油泵→漏油 |
| CAUSED_BY | Failure→Cause | 漏油→密封圈老化 |
| FIXED_BY | Failure→Solution | 漏油→更换密封圈 |
| REQUIRES_CAUTION | Solution→Caution | 更换密封圈→释放压力 |
